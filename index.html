Tag Cloud

What it is

Welcome to the Tweet Cloud. This web app uses Twitter to get tweets, and arranges them into an interactive three-dimensional sphere.

How it Works

You can move the cloud around with your mouse, and one tweet at a time will be highlighted. When you click inside the cloud, you will activate the highlighted tweet, which means:

  • If the the tweet contains a link, your browser will ask if you'd like to go to that page.
  • If the tweet has a picture, the page will display that picture.
  • If the tweet has neither, the text of the tweet will be shown.

Search Options

There are two kinds of Tweet Clouds you can create.
  1. On the left, you can make one using anyone's Twitter screen name.
    • Examples: Earth_Pics, HistoryInPics, BabyAnimalPics
  2. On the right, you can create one by searching for any term you desire.
    • Examples: NYC, SuperBowl, CrossFit
You can manually refresh the Feed Cloud with ten previous tweets, while the Search Cloud will automatically refresh with more tweets. Have fun!
