Tag Cloud

What it is

Welcome to the Tweet Cloud. This web app uses Twitter to get tweets, and arranges them into an interactive three-dimensional sphere.

How it Works

The tweet cloud responds to the movements of your mouse, and at any given time, one tweet will be highlighted depending on where your mouse is. When you click on the screen, your browser will display the text contained in the highlighted tag, and if the tweet comes with a link, the page will ask whether you'd like to travel to the URL.

Search Options

Currently, there are two kinds of tweet clouds you can create. With the search field on the left, you can create a tweet cloud with your Twitter screen name (or anyone else's too). On the right, you can create a cloud by searching Twitter for any term you desire. Tag clouds will display ten tweets at any given time: the Feed Cloud will allow you to manually refresh the cloud with ten previous tweets, while the Search Cloud will automatically refresh with more tweets. Have fun!
